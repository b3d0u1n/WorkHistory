Yodlee provided the backend scraping service, we provided account linking to a single Yahoo! ID. Users were able to link credit card accounts, brokerage accounts, domestic and foreign bank accounts. Yodlee provided a rolling history without unique transaction IDS. This was a thirty day view of transactions with three fields: transaction date, transaction description, amount. All three fields for a financial event could change in a subsequent crawl. For example, the date would advance after tip reconciliation, the amount would increase after tip reconciliation, and the description would often change as additional description was provided by the institution.
