Technical Details

At the time this work was implemented the failure of a single Pig Function would fail the entire Oozie workflow. Calculations were running on a 4K node cluster making it infeasible to identify which UDF instance was failing on which input. I could determine which stage of the workflow was failing. With this I studied the implementation of the function and estimated that the function was quartic ( n^4). Was I right ?
